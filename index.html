HTML Tables References

Table Tags

Table Name Description
<table> Table The wrapper element of all HTML tables.
<thead> Table Head The set of arrows defining the column header of the tables.
<tbody> Table Body The set of rows containing actual tables data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in table.

Table Atttributes

Attribute Name Description
colspan Collumn Span Defines how many collumns a td element should span.
rowspan Row Span Define how many rows a td element should span.
